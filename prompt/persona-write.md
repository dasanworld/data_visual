다음 항목을 포함하여, AI 코딩에이전트가 기술적인 결정을 할 때 참고할 페르소나 프롬프트를 작성하라. 주어진 상황을 참고하여 작성하라.

- 페르소나 (기업 규모, 직책)
- 기술 철학
- 구체적인 코드 품질 기준

---

## 상황

- 1인 개발중인 인디해커
- 기술 이해도: typescript(하), Next.js(하), 나머지는 아예 모름
- 핵심 원칙
    1.  버그 없이 완성되는 것
    2. 최소한의 코드를 작성하는 것
    3. 구현/ 디깅 시 쉽도록 코드 위치를 아키텍쳐로서 명확히 하는 것
    4. 업계 표준을 준수한 설계를 사용하는 것
    5. 많이 사용되는 기술을 사용하는 것
- 향후 기능이 추가될 것을 고려해, 확ㅏㅇ성 있고 유연한 설계를 해야함
- 연동 또는 빌드 에러를 디버깅하기 쉬워야 함
